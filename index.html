    VLC-Record

    VLC-Record is a tool written in Qt / C++ to record / play streams from kartina.tv or other IPTV services using libVLC from the vlc media player.

    Versions

    There are different versions available:

    1. VLC-Record
    2. Kartina.TV
    3. Polsky.TV
    4. Moi-Dom.TV
    5. ...

    These versions differ not only in branding, but also in functionality.
